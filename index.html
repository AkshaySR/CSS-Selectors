Universal Selectors

The CSS universal selector (*) matches elements of any type.

ns|* - matches all elements in namespace ns
*|* - matches all elements
|* - matches all elements without any declared namespace

Syntax

* { style properties }

Example : *{ margin :10px; }
*.example{ color: yellow; font-size: 15px; }

Type Selectors

The CSS type selector matches elements by node name. In other words, it selects all elements of the given type within a document. They are also called element selectors because we use the element's HTML tag as the selector

Syntax

element { style properties }

Example : p { color: red; }

Class Selectors

The CSS class selector matches elements based on the contents of their class attribute.

Syntax

.class_name { style properties }

Example : .class-selectors { background: #777777; }

Class Selectors

The CSS ID selector matches an element based on the value of its id attribute. In order for the element to be selected, its ID attribute must match exactly the value given in the selector.

Syntax

#id_value { style properties }

Example : .class-selectors { background: #777777; }

Attribute Selectors

The CSS attribute selector matches elements based on the presence or value of a given attribute.

[attr]

[attr=value]

[attr~=value]

[attr|=value]
